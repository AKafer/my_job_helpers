# my_job_helpers

# Программы помощники скачивания и обработки отчетов

## Описание

### Проект содержит три программы - 

### Пользователи смогут заходить на чужие страницы, подписываться на авторов и комментировать их записи.

### Автор уже может выбрать имя и уникальный адрес для своей страницы. Дизайн пока что самый простой.

### Записи можно отправить в сообщество и посмотреть там записи разных авторов.

## Как установить проект

### Клонировать репозиторий и перейти в него в командной строке:

```
git clone https://github.com/AKafer/SOSBLOG.git
cd SOSBLOG/
```

### Создать и активировать виртуальное окружение:

```
python -m venv venv
source venv/Scripts/activate
```

### Установить зависимости из файла requirements.txt:

```
pip install -r requirements.txt
```

### Выполнить миграции:

```
cd yatube
python manage.py migrate
```

### Запустить проект:

```
python manage.py runserver
```
